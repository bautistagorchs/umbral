# UMBRAL

Umbral es un proyecto web conceptual.

No propone una experiencia.
La acompaña.

Es una pausa | Un espacio intermedio.

---

## Sobre el proyecto

Umbral explora:
- relación entre tiempo, espacio y presencia
- valor del silencio visual
- decisión de diseño minimal

---

## Decisiones técnicas

- Next.js (App Router)
- React
- TypeScript

No librerías de UI.
No frameworks de CSS.

El diseño es propio.
Las animaciones son CSS puro.
Los videos son nativos.

La arquitectura es mínima.
Nada sobra.
